Introducción

El objetivo de este curso es introducir a todos los participantes al lenguaje de programación R, manejar diferentes variables, así como estructuras de control, finalmente comenzar el camino en el análisis geoestadístico.

Este manual ha sido elaborado en R con bookdown.El código del curso se encuentra en GitHub

Metodología

Actividades prácticas mediante ejercicios que refuercen el aprendizaje del tema. Los estudiantes aprenderán haciendo y utilizando el lenguaje de programación R y el entorno de desarrollo RStudio

Propósito

El curso se ha planteado de manera que cumpla con dos propósitos fundamentales:

  • Vencer el miedo a programar, y recordar siempre que los errores son solo eso, errores
  • Empezar a trabajr con datos y generar autonomía propia en codificación.
