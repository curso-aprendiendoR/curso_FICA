Capitulo8 Condicional IF - ELSE

La estructura if - else es un conjunto de dos par√°metros: la condici√≥n y la instrucci√≥n si la condici√≥n no se cumple. ESto nos permite tener dos indicaciones una cuando la condici√≥n si se cumple y una cuando no.

Vamos a realizar un ejercicio en donde ocuparemos el condicional IF - ELSE y una funci√≥n

operar<- function(a,b,operacion ="suma"){ # Esta llave abre la indicaci√≥n de la funci√≥n
  if(operacion == "suma"){ # Verdadero si operaci√≥n es igual a la palabra suma
    c<- a+b
  }
  else{# Falso si la palabra no corresponde a suma
    c<- NA
  }
  return(c)
}
operar(5,7,operacion="suma")
#> [1] 12
operar(5,7, operacion = "resta")
#> [1] NA

8.1 IF - ELSE anidados

La estructura de control if - else puede anidarse, es decir puede concatenarse para obtener instrucciones y acciones m√°s complejas

operacion_a_b<-function(a,b,operacion='suma'){
  if(operacion=='suma'){# Si operacion es igual a suma
    c<-a+b
  }else if(operacion=='resta'){ # Si operacion no es igual a suma y es igual a resta
    c<- a-b
  }else if(startsWith(operacion,'multi')) { # Si la operaci√≥n no es igual a resta y si cumple la condici√≥n
    c<-a*b
  }else{ # Si operacion no cumple la condici√≥n
    c<-NA
  }
  c
}

Hemos agregado al c√≥digo el comando startWith()que permite determinar una cadena de texto que comience con los caracteres indicados, veamos como funciona este comando y la estructura de control anidada

operacion_a_b(5,7,operacion='resta')
#> [1] -2
operacion_a_b(5,7,operacion='suma')
#> [1] 12
operacion_a_b(5,7,operacion='multiplicaremos')
#> [1] 35
operacion_a_b(5,7,operacion='xxmultiplicaremos')
#> [1] NA
operacion_a_b(5,7,operacion='multiplica')
#> [1] 35
#Declaremos las variables previamente - Es decir coloquemos informaci√≥n en las variables

b<-3

#> [1] "La suma de a + b es: 8"
paste('La resta de a - b es:',operacion_a_b(a,b,operacion='resta'))
#> [1] "La resta de a - b es: 2"

8.2 ifelse() Vectorizado

¬øQue significa vectorizar una estructura condicional? Cuando vectorizamos podemos construir en una sola fila un n√∫mero elevado de estructuras de condici√≥n, por lo que los argumentos de la funci√≥n ifelse() seguir√°n el orden: (condici√≥n, que sucede si se cumple, que sucede si no se cumple).

suma_resta<-function(a,b,operacion='suma'){
  ifelse(operacion=="suma",a+b,
         ifelse(operacion=='resta',a-b,NA))
}
suma_resta(5,9,operacion='resta')
#> [1] -4
suma_resta(5,9,operacion='mult')
#> [1] NA
